# Demo Outputs

This folder contains example outputs generated by the multimodal agent.

## Contents

- **test_image_output.png** - Sample image generated with Stable Diffusion XL
  - Prompt: "A serene mountain landscape at sunset with a lake"
  
- **test_video_output.mp4** - Sample video generated with text-to-video model
  - Prompt: "Ocean waves gently crashing on the shore"

## Generate Your Own

To create your own demos:

```bash
# Generate an image
python -m src.generators.text_to_image

# Generate a video
python -m src.generators.text_to_video

# Or use the web UI
python -m src.ui.web_ui
```
